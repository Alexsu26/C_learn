## 格式化输出
`printf("%[flags][width][.prec][hIL]type")`中
- **flags**
    - **-**:左对齐
    - **+**:前面强制输出+
    - **0**:前面填充0
- **width**
    - **number**：总共占据几位，包括小数点和小数部分
    - **.number**：小数部分占据几位
    - **\***：后面的第一个参数占据\*号的位置
    - **.\***：参考上面*

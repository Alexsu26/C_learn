## 编译预处理
> #include 不是用来引入库的，而是把`.h`头文件中所有的代码在编译是原封不动地代替到`main`中的#include命令行
真正的函数的源代码在另一个`.c`文件中
**在定义和使用头文件的地方都应有#include**
在函数前加`static`使得函数范围只在当前`.c`文件中能用
## 变量的声明
> `int i`是变量的定义  
`extern int i`是变量的声明，*不能初始化，不产生源代码*
## 标准头文件结构
> 为了防止同一个的结构在不同头文件多次定义，使用**标准头文件结构**
```C
#ifndef _LIST_H_
#define _LIST_H_
...
...
...
#endif

```
